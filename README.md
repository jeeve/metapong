# metapong

A small multiplayer Pong game running on a Nodejs server. The game space grows as users connect.

Install Nodejs
https://nodejs.org/

Download Git repository

At root direcrectory, enter :

`npm install`

`node server.js`

Navigate to http://localhost:8080
